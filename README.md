## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install bajo-web-mpa-fontawesome
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```bajo-web-mpa-fontawesome``` in it.
